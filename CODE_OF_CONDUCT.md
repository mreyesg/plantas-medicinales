# Code of Conduct

Todos los participantes en este proyecto deben respetar a los demás. Este repositorio adopta el Código de Conducta del Contribuyente (Contributor Covenant) para establecer normas claras de comportamiento.

Puntos clave:

- Trata a todos con respeto y cortesía.
- Evita lenguaje ofensivo, ataques personales y acoso de cualquier tipo.
- Sé colaborador y constructivo en las revisiones y discusiones.

Cómo reportar comportamientos inadecuados:

1. Si experimentas o ves un comportamiento que viole este código, abre un *issue* privado (si procede) o contacta a los mantenedores.
2. Incluye detalles del incidente, participantes, fechas y cualquier evidencia relevante.

Consecuencias:

Los mantenedores se reservan el derecho de tomar acciones (advertencia, eliminación de comentarios, bloqueo o remoción del proyecto) dependiendo de la gravedad de la infracción.

Esta política se aplica tanto en issues, pull requests, comentarios de código y canales públicos relacionados con el proyecto.
